# Ai Development Notes and Setup

## `CLAUDE.md`

Contents of this file are automatically pulled into the context when starting a conversation. 

- bash commands, 
- utility functions and core files
- code style
- testing instructions
- environment setpup
- behaviours and warnings about your project

### Location

1. root of repository (you can name this `CLAUDE.md` or `CLAUDE.local.md`)
2. parent and child directories of where you run claude (all files will be pulled in)
3. `~/.claude/CLAUDE.md` which applied to all claude sessions

### Permissions

You can save permissions to actions by using `.claude/settings.json` or `~/.claude.json`
